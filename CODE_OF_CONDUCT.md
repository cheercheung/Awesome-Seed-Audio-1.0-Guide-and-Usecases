# Code of Conduct

## Our Standard

Contributors should keep discussions focused, evidence-based, and respectful. This repository collects public Seed-Audio 1.0 use cases, so corrections and disagreements should cite sources rather than personal attacks.

Unacceptable behavior includes harassment, discriminatory language, spam, doxxing, and knowingly submitting fabricated sources or claims.

## Enforcement

Maintainers may edit or remove comments, issues, pull requests, or contributions that violate this standard. Repeated or severe violations may lead to blocked participation.

If you need to report a conduct issue, open a maintainer-directed issue with the minimum necessary detail, or use GitHub's reporting tools when the issue involves platform abuse.
